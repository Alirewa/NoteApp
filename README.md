## 📖 Overview

**Note App** is a fully client-side note-taking application built with modular Vanilla JavaScript. All notes are saved in `localStorage`, so your data persists between sessions — no server required. A great example of clean JavaScript architecture without any framework.

---

## ✨ Features

- ✅ Create, edit, and delete notes
- ✅ Pin important notes to the top
- ✅ Data persisted in `localStorage`
- ✅ Modular JavaScript architecture (ES Modules)
- ✅ Fully responsive layout
- ✅ Zero dependencies — no build step needed

---

## 🛠️ Tech Stack

| Technology | Role |
|---|---|
| JavaScript (ES6+ Modules) | Application logic & state management |
| HTML5 | Markup & structure |
| CSS3 | Styling & responsive layout |
| localStorage API | Client-side data persistence |

## 🚀 Getting Started

```bash
git clone https://github.com/Alirewa/NoteApp.git
cd NoteApp
# Use a local server for ES Modules support
npx serve .
```

Or use the **[live demo](https://note-app-alirewa.vercel.app)** directly.
